How are the CO2 numbers calculated?

Deterministically, in code - never by the AI. For commutes we take the real route distance from the Google Maps Routes API and multiply it by a fixed emission factor for each mode. For appliances we convert usage into daily kWh and multiply by the grid emission factor. The same inputs always produce the same numbers.

What emission factors do you use, and where are they from?

Commute (kg CO2 per km): driving 0.192 (average petrol car), bus/transit 0.105, walking 0, cycling 0.

Electricity: 0.71 kg CO2 per kWh, from the CEA CO2 Baseline Database, Version 21.0 (Nov 2025), FY 2024-25 weighted average. Appliance power ratings use standard BEE-rating ballpark figures.

What does the AI actually do?

The Gemini model only writes the natural-language explanation and answers your follow-up questions. It uses the numbers the code already computed - it does not calculate or change any carbon figure. If the model is unavailable, a built-in factual summary is shown instead.

How is the recommended option chosen?

Options are sorted greenest-first (lowest CO2, then fastest). The recommendation is the lowest-carbon option that is realistic for the distance: walking is suggested up to 3 km and cycling up to 10 km, while transit and driving are viable whenever a route exists. The recommended card is marked with a label and a border, not by colour alone.

What is the "this week" total?

It combines the CO2 you saved on logged commutes with the estimated CO2 from logged appliances for the current week, and also shows each part separately so the two are never confused.

Do you track my location?

No. There is no GPS or location permission. You type place names as plain text and the Maps Routes API geocodes them. The website only ever talks to the GreenRoute backend, which holds the API keys - your browser never sees a key.

How accurate is this?

These are good-faith estimates using average factors (for example, an average petrol car and average bus occupancy). Your real emissions depend on your specific vehicle, occupancy, and driving conditions, so treat the numbers as directional rather than exact.
